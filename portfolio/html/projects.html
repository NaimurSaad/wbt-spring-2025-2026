Projects

Urban City Life

Description: This project is a 2D urban city life animation developed using C++, OpenGL, and GLUT that represents a riverside environment inspired by real-life city surroundings. The scene features residential buildings, trees, a river with two kayaking boats moving in opposite directions, and a metro rail traveling horizontally across its track. It includes both Day and Night modes, where during the day the sun appears, boats move continuously, and a character flies a kite using keyboard controls, while at night the sky becomes dark, the moon appears, building windows light up, metro headlights turn on, and boat movement stops automatically to create a realistic nighttime atmosphere.

Skills Used: C++, OpenGL, GLUT, 2D Graphics, Animation, Object Movement, Keyboard Interaction
